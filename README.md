# Data-Analysis-of-Electron-Channel-of-the-Drell-Yan-Process-using-CMS-open-data
ROOT-based analysis of the Drell-Yan (Z→e⁺e⁻) process at the LHC. The workflow includes tree slimming, MET handling, electron selection cuts, and kinematic plotting. Designed to study signal purity and background rejection, reflecting a simplified CMS-style high-energy physics analysis pipeline.
